## Полезные настройки

### Prettier

    "editor.codeActionsOnSave": {
        "source.fixAll.eslint": true
    },
    "editor.formatOnPaste": true,
    "editor.formatOnType": true,
    "editor.wordWrap": "on",
  ```

        ["[", "]"],
        ["{", "}"],
        ["Gold", "Orchid", "LightSkyBlue"],
        "Red"
    ],
  ```
